# 俄罗斯方块

demo 链接：https://tetris.webgames.fun

实现了经典俄罗斯方块的核心玩法，包括方块下落、旋转、消除等功能。

设计：
- GridManager，维护 grid 数组，检测 grid 坐标内是否被占据、grid 边界，进行消除检查、消除等。
- Tetromino，下落中的方块，随机生成形状，处理渲染和移动，加速下落，落到最底部 store_block 把每个 block 交给 grid 接管。
- Renderer，根据 grid 数组渲染 TileMap（游戏背景网格）、Sprite（Block）。
- GameManager，控制游戏主循环，例如开始、暂停、结束等，并在消除完成后生成新的Tetromino

后续规则完善可参考：https://tetris.huijiwiki.com/wiki/%E4%BF%84%E7%BD%97%E6%96%AF%E6%96%B9%E5%9D%97%E5%87%86%E5%88%99

超级旋转系统：https://tetris.huijiwiki.com/wiki/%E8%B6%85%E7%BA%A7%E6%97%8B%E8%BD%AC%E7%B3%BB%E7%BB%9F

1. 项目使用 Godot 引擎开发
2. 主要使用 GDScript 编写游戏逻辑

## **操作说明**

- **键盘（PC / 浏览器）**:
	- **左右方向键**：将方块向左/向右移动；按住可连续移动。
	- **下方向键**：软降，按住可加速下落（每下落一格 +1 分）。
	- **上方向键 / 点击**：旋转方块（顺时针 90°）。
	- **空格键**：硬降，方块立即落到底（每下落一格 +2 分）。
	- **Esc（或暂停按钮）**：暂停/继续游戏。

- **手机触摸操作**:
	- **轻触 / 向上短滑**：旋转方块。
	- **向左 / 向右滑动**：将方块移动到对应方向；快速滑动可一次移动多格，长按可持续移动。
	- **向下缓慢滑动**：软降（每行 +1 分）。
	- **向下快速或大幅滑动**：硬降（双倍分数，每行 +2 分）。
	- **屏幕上的暂停按钮**：暂停/继续。

> 小提示：触摸输入包含速度与距离判断，快速滑动与长按会带来不同的移动行为，点击与向上滑动均可旋转。

## **等级与计分说明**

- **起始等级**：在开始菜单中可选择起始等级（1–10）。
- **等级提升**：每消除 10 行，等级提升 1 级（显示在界面上）。等级越高，方块下落速度越快，难度更大。

- **消行计分（基础分 × 等级）**：
	- 消除 1 行：40 × 当前等级
	- 消除 2 行：100 × 当前等级
	- 消除 3 行：300 × 当前等级
	- 消除 4 行（Tetris）：1200 × 当前等级

- **下落计分**：
	- 软降（手动加速下落）：每下落一格 +1 分（在方块锁定时结算）。
	- 硬降（立即落到底）：每下落一格 +2 分（硬降额外加成）。

- **界面提示**：屏幕上会显示当前分数、已消除行数、当前等级以及下一个方块的预览，方便玩家规划操作。
